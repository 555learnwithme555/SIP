OSPi

A highly cusomizable DIY irrigation/sprinkler controller based on Raspberry Pi

Download this project as a .zip file Download this project as a tar.gz file

Welcome to OSPi!

If you are interested in putting together a Raspberry Pi based irrigation (sprinkler) controller that can have custom software and hardware features you have come to the right place.

The core software is written in Python and the user interface, accessible via your favorite web browser, is created in JavaScript and jQuery.

If you are a programmer or would like to learn you can easily create plugins to add new software features or you can select from a growing number of useful plugins shared by others.
